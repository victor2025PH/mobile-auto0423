## 二、三视角分析

### 开发工程师视角

**核心问题:**
1. Instagram/Facebook是空壳(30%)，月活合计50亿的主战场开发投入最少
2. Vision引擎是"完美的未启用功能"，OmniParser没跑，靠硬编码坐标
3. TikTok 80+个选择器针对 Redmi 13C v43.8.3 硬编，APP更新就失效
4. compliance.yaml TikTok follow限制50/h(应该8)，上线会封号
5. 端到端流程未串联跑通过

### 市场经理视角

**核心问题:**
1. 没有可衡量的业务指标(触达数/回复率/转化率/线索成本)
2. 引流链路断裂(TikTok DM→Telegram→WhatsApp未打通)
3. 话术库空白，AI改写引擎做好了但没内容喂进去
4. 合规参数未经数据验证，不知道安全阈值
5. 与竞品(PhantomBuster等)比功能强但使用门槛极高

### 用户(操作员)视角

**核心问题:**
1. 424个API+30个面板，不知道第一步做什么
2. 想养号要写YAML+配API Key+理解工作流引擎，太复杂
3. 手机在干什么、到哪一步了、哪个账号出问题了 — 不直观
4. 出错不知道怎么办，告警模块有但没配规则
5. 看不到ROI，不知道赚没赚到
